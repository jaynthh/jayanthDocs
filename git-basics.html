Git & GitHub (Ultra Guide)

Learn Git visually + practically — from basics to advanced workflows.

What is Git?

Git tracks changes in your code and allows collaboration.

What is GitHub?

GitHub stores your Git repositories online.

Basic Commands

git init git status git add . git commit -m "message"

Branching (Important)

git branch feature git checkout feature git merge feature

Branches allow you to work without affecting main code.

GitHub Commands

git remote add origin URL git push -u origin main git pull git clone URL

Advanced (Interview Level)

git stash git rebase main git reset --hard HEAD git log --oneline

Real DevOps Workflow

git checkout -b feature git add . git commit -m "feature added" git push

💡 Pro Tip

Always use branches. Never work directly on main in real projects.
