Linux Basics

Start your Linux journey by understanding how the system works and how to interact with it using commands.

Level: Beginner Track: DevOps / Linux

What is Linux?

Linux is an open-source operating system used in servers, cloud platforms, and DevOps environments.

Most modern applications run on Linux servers.

Linux File System

Linux uses a hierarchical structure:

/ (root)
/home
/etc
/var
  • / → Root directory
  • /home → User files
  • /etc → Configuration files
  • /var → Logs and data

What is Terminal?

Terminal is where you run commands to interact with the system.

Linux is command-driven — most tasks are done using CLI.

Basic Commands

Check current location:

pwd

List files:

ls

Change directory:

cd /home

File Permissions (Basic Idea)

-rwxr-xr--

This represents read, write, and execute permissions for users.

Real-World Use

Linux is used to manage servers, deploy applications, and run backend systems.

If you want DevOps → Linux is mandatory.

💡 Real Tip

Don’t memorize commands — practice them daily.
