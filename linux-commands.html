Linux Commands (Pro Guide)

Master essential Linux commands used in real-world IT support and DevOps environments.

Level: Beginner → Intermediate Track: DevOps

File & Directory Commands

Navigation
pwd ls cd /home
File Management
touch file.txt mkdir folder rm file.txt

View & Read Files

cat file.txt less file.txt head file.txt tail file.txt

Permissions

chmod 755 file.sh chown user file.txt

Process Management

ps top kill -9 1234

Network Commands

ping google.com ifconfig netstat -tuln

System Info

df -h free -m uname -a

Real DevOps Workflow

cd /app ls cat config.txt ps top

This flow helps you navigate, check files, and monitor running processes.

💡 Pro Tip

Don’t try to learn all commands at once. Focus on categories and practice daily.
