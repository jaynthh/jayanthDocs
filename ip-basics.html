IP Address Basics

Understand what an IP address is, its types, and how it is used in real-world networking and troubleshooting.

Level: Beginner Track: Networking

What is an IP Address?

An IP (Internet Protocol) address is a unique identifier assigned to each device on a network.

It allows devices to communicate with each other over a network or the internet.

Example of IP Address

192.168.1.10

This is a typical IPv4 address used in local networks.

Types of IP Addresses

  • Private IP: Used inside local network (home/office)
  • Public IP: Assigned by ISP for internet access

Private IP Ranges

192.168.x.x
10.x.x.x
172.16.x.x – 172.31.x.x

⚠️ Special IP (Important)

169.254.x.x

This indicates the system failed to get an IP from DHCP (common network issue).

How to Check IP Address

Windows:

ipconfig

Linux:

ifconfig
or
ip a

Real-World Use (L1 Support)

When user reports "No Internet", checking IP is the first step. 

If IP = 169.x.x.x → DHCP issue
If no IP → network disconnected
If valid IP → check DNS or gateway

💡 Real L1 Tip

Always verify IP before doing advanced troubleshooting. Most issues can be identified just by looking at the IP.
