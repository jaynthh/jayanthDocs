HTML Images – Adding Images to Your Web Pages

Learn how to display images, control size, add alt text, and make responsive images.

Level: Beginner Estimated Time: 10 min Track: Web Development

The <img> Tag

HTML uses the <img> tag to display images. It is a self-closing tag (no ending tag required). 

<img src="photo.jpg" alt="Description of image">
  • src → path or URL of the image
  • alt → text shown if image fails to load (important for SEO & accessibility)

Image Size (width & height)

You can control image size using width and height attributes: 

<img src="car.png" width="300" height="200" alt="Car">

Recommendation: Use only width so the browser maintains correct aspect ratio.

Responsive Images (Fit in All Screens)

Make images automatically resize on mobile using CSS:

img {
  max-width: 100%;
  height: auto;
}

This ensures the image never overflows the screen.

Image from a Website URL

<img src="https://picsum.photos/400" alt="Random image">

Perfect for testing when you don’t have local images.

Clickable Image (Image Link)

Wrap the image inside an anchor tag:

<a href="https://google.com">
  <img src="logo.png" width="120" alt="Google">
</a>

Image Tooltip

Use the title attribute:

<img src="sun.png" alt="Sun" title="This is the sun">

Common Image Formats

  • JPG – Photos, high detail
  • PNG – Transparent backgrounds
  • GIF – Animation
  • SVG – Logos, icons (scalable)
  • WEBP – Modern, lightweight, recommended
