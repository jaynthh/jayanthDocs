Linux Basics

Learn how Linux works and the essential commands used in IT support and DevOps environments.

Level: Beginner Track: DevOps / Linux

What is Linux?

Linux is an open-source operating system widely used in servers, cloud, and DevOps environments.

Why Linux is Important

  • Used in most servers
  • Required for DevOps
  • Fast and secure

Basic Commands

Check current directory:

pwd

List files:

ls

Change directory:

cd /home

Create file:

touch file.txt

Example Workflow

cd /home ls touch test.txt

This sequence moves to a directory, lists files, and creates a new file.

Real-World Use

In DevOps and IT support, Linux is used to manage servers, deploy applications, and monitor systems.

💡 Real Tip

Focus on practicing commands daily. Linux is learned by doing, not reading.
