HTML Text & Links

Learn headings, paragraphs, and how to create links between your pages.

Level: Beginner Estimated Time: 15 min Track: Web Development

Headings & Paragraphs

HTML provides six heading tags from <h1> to <h6>... 

<h1>Main Title</h1>
<h2>Section Title</h2>
<p>This is a paragraph of text.</p>

Links

Use the <a> tag to create links:

<a href="https://jayanthb.in">Visit my site</a>

To link between your tutorial pages:

<a href="index.html">Back to Introduction</a>
