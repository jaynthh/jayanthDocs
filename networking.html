Networking Basics

Understand how systems communicate and how to troubleshoot network issues in real-world IT support.

Level: Beginner Track: IT Support / Networking

What is Networking?

Networking is the process of connecting computers and devices so they can communicate and share data.

In IT support, networking knowledge helps you diagnose issues like no internet, slow connections, and connectivity failures.

Key Networking Components

  • IP Address: Unique identifier for each device
  • DNS: Converts domain names to IP addresses
  • Router: Connects network to the internet
  • Switch: Connects devices inside a network

How Internet Works (Simple Flow)

  1. You enter a website (google.com)
  2. DNS converts it to an IP address
  3. Request goes through router
  4. Server responds with data

Common Network Issues

  • No internet connection
  • Slow network
  • DNS not resolving
  • IP conflict

💡 Real L1 Tip

Always check basics first: cable, Wi-Fi, IP address. Most network issues are simple but look complex.
