How to Fix No Internet Issue

Step-by-step troubleshooting used by L1 engineers to quickly restore network connectivity.

Level: Beginner Time: 10–15 min Scenario: Network Issue

🔍 Problem

User reports: "Internet is not working"

⚡ Quick Checks (Always Start Here)

  1. Check Wi-Fi / LAN connected
  2. Check airplane mode OFF
  3. Restart router & system
  4. Try another device (to confirm issue)

🧠 Step-by-Step Troubleshooting

1. Check IP Address

ipconfig

If IP starts with 169.x.x.x → network issue (no DHCP)

2. Renew IP Address

ipconfig /release
ipconfig /renew

3. Check Connectivity

ping 8.8.8.8

If ping fails → network problem

4. Check DNS

ping google.com

If IP ping works but domain fails → DNS issue

5. Reset Network

netsh int ip reset
netsh winsock reset

🛠 Common Causes

  • Router issue
  • IP conflict / DHCP failure
  • DNS problem
  • Network adapter issue

💡 Real L1 Tip

Always check if issue is user-specific or network-wide. If multiple users are affected → escalate quickly.

✅ Resolution Note (For Ticket)

Internet restored after renewing IP address and resetting network stack. User confirmed connectivity is working.
