How to Fix a Slow PC

Simple step-by-step methods used by L1 support engineers to improve system performance.

Level: Beginner Time: 10–15 min Scenario: Real Support Ticket

🔍 Problem

User reports: "My system is very slow"

⚡ Quick Fix Steps (L1 Method)

  1. Restart the system
  2. Close unused applications
  3. Check Task Manager for high usage
  4. Disable startup programs
  5. Run disk cleanup

🧠 Step-by-Step Solution

1. Open Task Manager

Ctrl + Shift + Esc

Check CPU, Memory, Disk usage and end unnecessary tasks.

2. Disable Startup Apps

Task Manager → Startup → Disable unwanted apps

3. Run Disk Cleanup

cleanmgr

4. Check Disk Space

Ensure at least 20% free space in C drive.

5. Restart System

Simple but solves many performance issues.

🛠 Advanced Checks (If Still Slow)

Check for Windows Updates
Scan for malware
Upgrade RAM (if usage > 80%)
Check disk health (HDD vs SSD)

💡 Real L1 Tip

Always start with basic steps before escalating. In most cases, slow PC issues are resolved without escalation.

✅ Resolution Note (For Ticket)

System performance improved after disabling startup apps and clearing temporary files. User confirmed issue resolved.
