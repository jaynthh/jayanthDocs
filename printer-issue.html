Printer Not Working

Step-by-step troubleshooting used by L1 engineers to fix printing issues quickly.

Level: Beginner Time: 10–15 min Scenario: Office Printer Issue

🔍 Problem

User reports: "Printer is not printing"

⚡ Quick Checks

  1. Check printer power ON
  2. Check cable / Wi-Fi connection
  3. Check paper & ink levels
  4. Set correct default printer

🧠 Step-by-Step Fix

1. Check Printer Status

Go to: Control Panel → Devices & Printers

Make sure printer is Online

2. Restart Print Spooler

services.msc

Find Print Spooler → Restart

3. Clear Print Queue

Cancel all stuck print jobs

4. Reinstall Printer

Remove device → Add printer again

5. Check Driver

Update or reinstall printer drivers

🛠 Common Causes

  • Print spooler stopped
  • Driver issue
  • Network printer not reachable
  • Stuck print queue

💡 Real L1 Tip

If multiple users cannot print → likely network printer issue. If only one user → local system issue.

✅ Resolution Note (For Ticket)

Printer issue resolved after restarting print spooler and clearing queue. User confirmed printing is working.
